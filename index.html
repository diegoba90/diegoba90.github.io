What is Computer Science?

If you enjoy to stream movies, play video games, work, study or just plain surf the web but wondered how all that works then Computer Science might be the degree for you. Computer Science is the study of computers, including but not limited to computational theory, hardware and software design, algorithms, and the way humans interact with computers

If you are interested in a challenging but yet rewarding path with plenty of room to grow then you might want to consider becoming a Computer Scientist. To learn more visit our Program Description page!

Different Concentrations

The different concentrations you can learn at AUM are:

  • Full Stack and Mobile Application Development
  • Cyber Security
  • Computer Animation and Game Development
  • Artificial Intelligence and Robotics Engineering
  • Big Data Analytics and Distributed Computing

Programming Languages

Here at AUM, you will have the potential of learning the following programming languages:

  • C++
  • Swift
  • Python
  • Java
  • Web Dev: HTML5, CSS3, JavaScript, JQuery
