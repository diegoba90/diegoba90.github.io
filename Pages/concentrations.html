Different Concentrations

Full Stack and Mobile Application Development

Build complete and immersive web and hybrid mobile applicatons. Master front and back end web, hybrid mobile applications and server-side development


Cyber Security

Learn to secure everything you do on computer systems. You will learn network security, application security, information security, operational security and disaster recovery and business contnuity.


Computer Animation and Game Development

Learn how to animate and create video games. From game programming to 3D structures and modeling and motion capturing, this program has everything you need to get you into one of the hottest fields in Computer Science


Artificial Intelligence and Robotics Engineering

This concentration will focus mainly on the studies of theory, algorithms, and systems for making intelligent decisions in a complex and uncertain environments.


Big Data Analytics and Distributed Computing

Big data technologies are used to achieve any type of analytics in a fast and predictable way enabling better decision making. Here you will learn the basics of data sotrage, data access, data transfer, visualization and predictive modeling.
